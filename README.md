# SQL_Hospital_Appointment_and_Managemen_System
1.Designed and developed a database system for managing patients, doctors, appointments,        prescriptions, and billing records.

2.Created and maintained 5 relational tables using Primary Keys and Foreign Keys. Implemented relationships between tables using INNER JOIN, Subqueries, and Aggregate Functions.

3.Generated reports such as:
   1.Patient with the highest payment
   2.Total hospital revenue
   3.Most frequently prescribed medicines
   4.Doctor-wise revenue and appointment reports
